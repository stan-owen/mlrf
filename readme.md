## How to use ?
### Dependencies
- `pip install -r requirements.txt`
### Data
- Please download the dataset from this [page](http://www.cs.toronto.edu/~kriz/cifar.html)
- Make sure to unzip the dataset in the `data` folder
### Run
- `python src/main.py`
